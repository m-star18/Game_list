# Spielliste

"Spielliste' is a emulator management list.

When you write game information to a file, it is displayed.

# Requirement
 
* Python 3.7.5
* Pillow 6.1.0
* PySimpleGUI 4.14.0
 
# Installation
 
```bash
pip install Pillow
pip install pysimplegui
pip install saves
```
 
# Usage
 
```bash
git clone https://github.com/m-star18/Spielliste
cd Spielliste
python main.py
```
